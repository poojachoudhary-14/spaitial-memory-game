## 🧠 The Cognitive Science Behind CosmicRecall

In cognitive psychology and neuroscience, this game is a digital adaptation of the **Visual-Spatial Working Memory Task** (frequently measured using the **Corsi Block-Tapping Task**). 

The game tests and trains your **Visual-Spatial Sketchpad**—the component of working memory responsible for temporarily storing and manipulating visual and spatial information.

---

## ✨ Key Features

* **Dynamic Grid Cover Mechanic:** First, only a few blocks flash to show you the target cells. Suddenly, new "distractor" blocks scale in with elastic CSS animations to completely cover the grid, hiding the solution and forcing you to use spatial memory mapping.

* **Procedural Sound Engine:** No external audio assets required! It features 100% code-synthesized feedback (chimes, ticks, buzzers, and level-up arpeggios) using the **HTML5 Web Audio API**.

* **Forgiving Lives System:** You start with 3 lives. Making a mistake reveals the solution briefly and resets the current level, giving your brain a chance to self-correct rather than instantly kicking you out.

* **Seamless Persistence:** Saves your personal high score locally using the browser's `localStorage` API.

* **Ultra-Modern Cosmic Styling:** Developed with a responsive Tailwind CSS utility framework and customized with sleek radial gradients, backdrop blurs, glow effects, and modern fluid typography.
